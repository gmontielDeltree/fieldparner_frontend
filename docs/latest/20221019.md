## NDVI -> Nuevos Indices Espectrales
La seccion de NDVI ahora se llama "Indices espectrales" y contiene los indices
NDVI
ReCL Índice de clorofila de borde rojo.
NDRE Diferencia Normalizada del Borde Rojo.
MSAVI Índice de Vegetación Ajustado al Suelo Modificado.
NDMI Índice de Humedad de Diferencia Normalizada.

## "Client Side" Routing 
Ahora es posible acceder a algunas (iré agregando mas) secciones del sitio directamente usando URLs.
Es util para que el usuario pueda agregar marcadores de secciones especificas (eventualmente compartir?). 
Tambien sirve para que en el uso en celular el boton 'BACK' devuelva a la vista anterior como en una app nativa

URL's por ahora:
/campos -> lista de campos
/indices/{uuid del lote} -> seccion de índices.
